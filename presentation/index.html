Fake Bitcoin QR Generator

Presentation by: Adeem Mawani

Background Info

  • Today, it's a common practice to share a Bitcoin address as a QR code
  • The user scans the QR code with a Bitcoin wallet app and sends the requested payment without having to type a lengthy Bitcoin addresses by hand
  • This eliminates the possibility of mistyping the address, but poses it's own security risks

Malicious Services

  • Last week, Harry Denley, Director of Security at the MyCrypto platform stumbled across many malicious services claiming to convert Bitcoin addresses into QR codes
  • He found a total of 9 sites, all sharing the same user interface, so it is likely they were all created by the same scammer

Malicious Services Cont.

  • So how were they malicious in nature?
  • For each Bitcoin address entered, the site always generated the same QR code -- for a scammer's wallet
  • This meant that whenever the code was used, the money was sent to the scammer's address instead of the users

Scammer's Adress

  • The scammer's address received funds totaling more than 7 BTC (~$45,000 USD)
